Implementation Details

  • Font Source: Google Fonts
  • Primary Font: Ubuntu (with system font fallbacks)
  • Font Weights: 300 (Light), 400 (Regular), 500 (Medium), 700 (Bold)
  • Font Display: swap (for optimal performance)
  • Fallback Fonts: System fonts and sans-serif
